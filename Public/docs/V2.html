OrderManV14 🚀

Modern Order Management Project with schema-based data generation, dynamic CRUD UI creation, modular Express.js APIs, and KeshavSoft VS Code extension integration.

Node.js Express.js CRUD UI Schema Generator

🛠️ Project Setup Workflow

1️⃣ Clone OrderManV14 repository from GitHub
https://github.com/keshavsoft/OrderManV14
2️⃣ Run Data file generator:
node Scripts/generateDataFromschemas.js
3️⃣ Install dependencies:
npm install

# if want to create index.html with header only or index.html with show table

4️⃣7 Create empty folder inside Public ex: -v1
5️⃣ Right click folder → Select ShowHtml
6️⃣ KeshavSoft Extension two options
📄

Header Only

Generate only navigation header UI inside HTML file. Best for header and layout initialization.

📋

With Header

Generate navigation header with CRUD table Show using provided table name.

7️⃣ run server:
npm nun start
8️⃣ open in browser:
open in browser
